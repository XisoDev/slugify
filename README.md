# XisoDev/slugify

> 한글을 포함한 문자열을 URL에서 사용 가능한 슬러그로 변경합니다.

## 주요한 사항

- cocur/slugify를 Fork한 프로젝트로, 오직 한글지원을 위해 만들어졌습니다.

## 설치

[Composer](https://getcomposer.org)를 통해 설치할 수 있습니다.

```shell
$ composer require xisodev/slugify
```
